# Component Usage Guide

This is a quick reference for all the components in your project.

## Available Components

### 1. PixelatedCanvas (Currently Active)
**Location:** `components/pixelated-canvas-demo.tsx`

```tsx
import PixelatedCanvasDemo from "@/components/pixelated-canvas-demo";

<PixelatedCanvasDemo />
```

### 2. BackgroundRippleEffect
**Location:** `components/background-ripple-effect-demo.tsx`

```tsx
import BackgroundRippleEffectDemo from "@/components/background-ripple-effect-demo";

<BackgroundRippleEffectDemo />
```

### 3. FeaturesSectionDemo (with Globe)
**Location:** `components/features-section-demo-3.tsx`

```tsx
import FeaturesSectionDemo from "@/components/features-section-demo-3";

<FeaturesSectionDemo />
```

### 4. AppleCardsCarousel
**Location:** `components/apple-cards-carousel-demo.tsx`

```

### 6. LoaderOne

```

## Quick Start

Your `app/page.tsx` is now set up with the PixelatedCanvas demo. To try other components:

1. Replace the import in `app/page.tsx`
2. Replace `<PixelatedCanvasDemo />` with your chosen component
3. Save and view in browser

## Running the Project

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000)
